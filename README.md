# Работа с git
### Скачать проект:
```bash
git clone <URL>
```
### Учесть все файлы:
```bash
git add .
```
### Сохранить коммит:
```bash
git commit -m "..."
```
### Отправка в облако:
```bash
git push
```
### Принять изменения из облака :
```bash
git pull
```
